# TextMeshPro WYSIWYG text editor Window
Unity EditorWindow to edit text with TextMeshPro tags with preview

# Installation
- drop TmpWysiwygUiElementsWindow.cs-file to Assets/.../Editor/ folder

# Features
- EditorWindow
- Ctrl + Z / undo
- preview inside EditorWindow
- select clean text block -> apply/clear any effects
- clear style button
- intersecting styles
- final result text to copy/paste
- many [TextMeshPro tags](https://docs.unity3d.com/Packages/com.unity.textmeshpro@4.0/manual/RichText.html)
- tag parameters
